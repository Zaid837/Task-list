#TASK LIST

This is a simple task list (todo list) used to save Your daily tasks in order not to miss anything.

FEATURES

- Add new task
- Mark task as completed / uncompleted
- Edit task
- Delete single task
- Delete completed tasks
- Clear all tasks

Technologies used:

- HTML
- CSS
- JavaScript/jQuery 3.1.1
